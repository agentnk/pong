# Pong Game

## Features
- Classic two-dimensional Pong gameplay
- Local multiplayer for two players
- Adjustable paddle sizes and ball speed
- Scoring system with visual output

## How to Play
Players control paddles on the left and right sides of the screen, trying to bounce the ball back and forth. The goal is to score points by getting the ball past the opponent's paddle.

## Game Mechanics
- **Paddle Movement**: Players can move their paddles up and down using control keys.
- **Scoring**: A point is scored when one player fails to return the ball.
- **Collision Detection**: The game detects collisions between the ball and paddles as well as the boundaries of the game area.

## File Structure
- `index.html` - Main HTML file that contains the game setup.
- `styles.css` - CSS file for styling the game elements.
- `script.js` - JavaScript file that contains the game logic and mechanics.

## Technical Details
This game is built using HTML5, CSS3, and JavaScript. The game utilizes the `<canvas>` element for rendering.

## Getting Started Instructions
1. Clone the repository: `git clone https://github.com/agentnk/pong`
2. Open `index.html` in your browser to start the game.

## Controls Summary
- Player 1: Use 'W' to move up, 'S' to move down.
- Player 2: Use 'Up Arrow' to move up, 'Down Arrow' to move down.

## Tips for Playing
- Keep an eye on the ball's speed to anticipate its movement.
- Practice controlling the paddle to improve reaction time.
